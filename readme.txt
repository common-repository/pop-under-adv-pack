== Description == 

The name tells it better than we can. It is a simply and powerfull pop-under manager plugin to edit, manage and take control of your pop-under ads. 
See the screenshot to understand how it works.

== Installation ==

1. Upload `popunder dir` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to POPUnder menu and start editing.
